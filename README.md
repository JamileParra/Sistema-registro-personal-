## 📌 DESCRIPCIÓN

Este proyecto consiste en el desarrollo de un sistema de registro de personal utilizando Python, aplicando los principios de la Programación Orientada a Objetos (POO).

El sistema permite gestionar empleados mediante un menú interactivo en consola, facilitando operaciones como registro, consulta, actualización y eliminación.

---

## 🎯 OBJETIVO

Desarrollar una aplicación funcional que permita administrar empleados utilizando buenas prácticas de programación y el paradigma orientado a objetos.

## 🏗️ ESTRUCTURA DEL SISTEMA

### 🔹 Clase `Empleado`

Representa a cada empleado con atributos como ID, nombre, edad y puesto.

### 🔹 Clase `SistemaPersonal`

Gestiona los empleados permitiendo agregar, buscar, listar, actualizar y eliminar.
